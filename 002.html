何を学ぶかって大事

エンジニアに興味がある、なってみたい。
という話はよく聞くものの、プログラマになって何がしたいかというと
まだよくわからないってパターン、多いですね。

これって料理人になりたいけど何作りたいかわからない。
スポーツ選手になりたいけど、何のスポーツやるかわからない。
というような状況です。

ここでは、うぇぶくえの狙いや、うぇぶくえをやると何が身につくのか
整理しながら何をやるかを明確にしていきます。

うぇぶくえやるとどうなるの？

うぇぶくえはあくまでもWEBがテーマ

なのでブラウザで動くもの
つまりはウェブサイトやウェブページが作れるようになります。

ウェブサイトとウェブページ

うぇぶくえはウェブページ作成が主体

ウェブページ
ページ単体。簡単なものから難しめのものまであるが比較的簡単。
一人で作る事が多い。
ウェブサイト
ウェブページが集まって出来たもの。
規模にもよるけど、一人で全て作る事は稀。

道具と材料と経験の話

ウェブページを作るためには
作るための道具と材料と経験が必要になります。

他の職種にも例えながら道具と材料と経験について整理する。

道具

WEBエンジニアの道具、それはうぇぶくえで扱っている

  • HTML
  • CSS
  • JavaScript

なんかがまさしく道具です。

料理人で言えば、包丁や鍋、火などに該当します。

材料

WEBエンジニアにとっての材料。

  • 画像
  • 動画
  • などなど...

料理人にとっての肉、魚、野菜などの食材に該当します。

経験

経験は特に説明する事もないですが以下の通りです。

WEBエンジニア
沢山のウェブページを作る
料理人
沢山の料理を作る

※必要なことは道具の使い方だけじゃない。

道具、材料、経験、大事なのは経験

道具は経験の中で磨かれる

道具の使い方だけじゃだめ、使って何をするかを考える

エンジニアを目指すとき、何かしらの言語を学ぶ必要はありますが
どう使うかもわからずに学んだところで効果が薄い。

考え、調べ、作り上げる経験

どうすれば作れるか、作るために何が必要か
常にそれを意識することで、仕事で使える実力を身につける事ができる。

うぇぶくえの狙い

うぇぶくえが道具の使い方に関して、あえて丁寧な解説をしない理由。

  • 考える経験
  • 調べる経験
  • クリアする経験

こういった経験を沢山積む事、それがうぇぶくえの一番の目的だから。
