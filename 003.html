HTMLとCSSとJavaScript

このサイトで学習するHTMLとCSSとJavaScript
それぞれが一体何なのか、どんな役割なのか
そんな話をします。

超ざっくりまとめると

  • HTMLは体(基本の土台)
  • CSSは洋服(おしゃれな見た目)
  • JavaScriptは動作(動きをつける)

ざっくり過ぎるので実際に見てみる。

HTML

HTMLでボタンを作る。(土台を作る)

CSS

CSSでおしゃれにしてみる。(見た目)

JavaScript

JavaScriptでボタンが押されたときに背景の色が変わるようにしてみる。
押すたびに色々な色になるよ！

今回はボタンが押されたときに、背景色を変えるという動作をつけました。
他にもボタンの上にマウスが来た時、キーボードのキーが押されたときなど
いろんなタイミングで、いろんな動作を付けられるのがJavaScriptです。

補足

HTMLは土台と書きましたが、本来のHTMLの役割は情報の構造化です。
そう聞くと難しそうと感じるかもしれませんが

  • このページにはこんな情報が含まれてますよー
  • このタイトルで検索できるようにしてねー
  • 全体の構成はこんな感じになってるよー
  • ここらへんが主張したいとこです！

というように文章だけではわからない情報を付与する。
それがHTMLの本来の役割です。
